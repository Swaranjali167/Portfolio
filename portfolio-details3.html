Safeify- Password Safety Web-Application

Safeify project is designed to create a system to save passwords safely. A user has to remember a lot of passwords for different accounts due to which they use the same password repeatedly for all the accounts. To help the user with these problems our system will help to store and organise their passwords. This system is designed in such a manner that it will suit all the needs of the user for keeping their passwords safe. The system has many advantages, but the main one is that the user needs to remember only a master password to access all the other passwords. This system is easy to use, user friendly and also free of cost.
